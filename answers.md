# Django Assignment Answers

## Question 1

### manage.py

`manage.py` is the command-line utility used to manage the Django project, such as running the development server, creating migrations, and executing administrative tasks.

### settings.py

`settings.py` contains the project's configuration settings, including installed apps, database settings, middleware, templates, and security options.

### urls.py

`urls.py` defines the URL routing for the project by mapping URL patterns to their corresponding views.
